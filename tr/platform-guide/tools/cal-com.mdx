## Cal.com Kurulumu

1. [Cal.com](https://cal.com/) web sitesine gidin ve bir hesap oluşturun
2. Yeni bir etkinlik türü oluşturun, örneğin "Yapay zeka asistanı ile toplantı"
3. Etkinlik türüne tıklayarak ve URL'den bu numarayı kopyalayarak etkinlik kimliğini kopyalayın

<img height="200" src="/resources/cal-event-id.png" />

4. Cal.com'da bir API anahtarı oluşturun

## Aracı asistanınıza ekleyin

Bu aracı asistanınıza ekleyin ve etkinlik kimliği ile API anahtarını girin.
