---
title: "WhatsApp Gönderenleri Getir"
api: "GET /user/whatsapp/senders"
description: "Kimliği doğrulanmış kullanıcının tüm WhatsApp Business gönderenlerini listeleyin"
---

Bu endpoint, hesabınız için yapılandırılmış WhatsApp Business gönderenlerinin (telefon numaraları) listesini döndürür. Bunu mesaj göndermek için gereken gönderen kimliklerini almak için kullanın.

### Sorgu Parametreleri

<ParamField query="status" type="string" optional>
  Gönderenleri duruma göre filtreleyin. Varsayılan: `online`. Durumdan bağımsız olarak tüm gönderenleri döndürmek için `all` kullanın.
</ParamField>

### Yanıt Alanları

<ResponseField name="data" type="array">
  <Expandable title="Gönderen nesnesi özellikleri">
    <ResponseField name="id" type="integer">
      WhatsApp göndereninin benzersiz tanımlayıcısı. Mesaj gönderirken bu kimliği kullanın.
    </ResponseField>

    <ResponseField name="phone_number" type="string">
      Gönderenin E.164 formatındaki telefon numarası (ör. `+14155551234`)
    </ResponseField>

    <ResponseField name="display_name" type="string">
      WhatsApp Business görünen adı (alıcılara gösterilen işletme adı)
    </ResponseField>

    <ResponseField name="status" type="string">
      Gönderenin mevcut durumu: `online` veya `offline`
    </ResponseField>

    <ResponseField name="quality_rating" type="string">
      Meta'nın bu gönderen için kalite değerlendirmesi: `GREEN`, `YELLOW` veya `RED`
    </ResponseField>

    <ResponseField name="messaging_limit" type="integer">
      Mevcut mesajlaşma limiti (24 saat başına benzersiz alıcı sayısı). `null` sınırsız anlamına gelir.
    </ResponseField>

    <ResponseField name="messaging_limit_formatted" type="string">
      Okunabilir mesajlaşma limiti (ör. `1,000 / 24hr`, `10,000 / 24hr` veya `Unlimited`)
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://app.whattalk.ai/api/user/whatsapp/senders" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash Tüm gönderenleri filtrele
curl -X GET "https://app.whattalk.ai/api/user/whatsapp/senders?status=all" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://app.whattalk.ai/api/user/whatsapp/senders',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const data = await response.json();
console.log(data.data); // Array of senders
```

```python Python
import requests

response = requests.get(
    'https://app.whattalk.ai/api/user/whatsapp/senders',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

senders = response.json()['data']
for sender in senders:
    print(f"{sender['display_name']}: {sender['phone_number']}")
```

### Notlar

- Varsayılan olarak yalnızca `online` durumundaki gönderenler döndürülür. Çevrimdışı gönderenleri dahil etmek için `?status=all` kullanın.
- Gönderen durumu her 5 dakikada bir Meta ile otomatik olarak senkronize edilir.
- `id` alanı, mesaj gönderirken `sender_id` olarak aktarmanız gereken değerdir.
- Kalite değerlendirmesi mesajlaşma limitlerini etkiler. `RED` değerlendirmesi mesaj gönderme yeteneğinizi kısıtlayabilir.
