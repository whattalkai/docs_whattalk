---
title: "Konuşmaları Listele"
api: "GET /user/conversations"
description: "Kimliği doğrulanmış kullanıcının tüm konuşmalarını filtreleme ve sayfalama ile listeleyin"
---

Bu endpoint, kimliği doğrulanmış kullanıcının asistanlarına ait sayfalanmış konuşma listesini döndürür. Bunu konuşma geçmişini görüntülemek, türe göre filtrelemek veya CRM'inizle entegre etmek için kullanın.

<Note>
  Bu endpoint kimlik doğrulaması gerektirir. API anahtarınızı `Authorization` başlığında Bearer token olarak gönderin.
</Note>

### Sorgu Parametreleri

<ParamField query="type" type="string" optional>
  Konuşmaları türe göre filtreleyin. Olası değerler: `test`, `widget`, `whatsapp`, `api`
</ParamField>

<ParamField query="assistant_id" type="integer" optional>
  Konuşmaları asistan kimliğine göre filtreleyin (kimliği doğrulanmış kullanıcıya ait olmalıdır)
</ParamField>

<ParamField query="date_from" type="string" optional>
  Bu tarihten itibaren konuşmaları filtreleyin (YYYY-MM-DD formatı)
</ParamField>

<ParamField query="date_to" type="string" optional>
  Bu tarihe kadar konuşmaları filtreleyin (YYYY-MM-DD formatı)
</ParamField>

<ParamField query="per_page" type="integer" optional>
  Sayfa başına konuşma sayısı (1-100, varsayılan: 15)
</ParamField>

<ParamField query="page" type="integer" optional>
  Sayfa numarası (varsayılan: 1)
</ParamField>

### Yanıt Alanları

<ResponseField name="data" type="array">
  <Expandable title="özellikler">
    <ResponseField name="id" type="string">
      Konuşmanın benzersiz UUID tanımlayıcısı
    </ResponseField>

    <ResponseField name="assistant_id" type="string">
      Bu konuşmayı yöneten asistanın UUID'si
    </ResponseField>

    <ResponseField name="assistant_name" type="string">
      Bu konuşmayı yöneten asistanın adı
    </ResponseField>

    <ResponseField name="type" type="string">
      Konuşma türü: `test`, `widget`, `whatsapp` veya `api`
    </ResponseField>

    <ResponseField name="message_count" type="integer">
      Konuşmadaki toplam mesaj sayısı
    </ResponseField>

    <ResponseField name="total_cost" type="number">
      Konuşmanın USD cinsinden toplam maliyeti
    </ResponseField>

    <ResponseField name="ai_enabled" type="boolean">
      Bu konuşma için yapay zeka yanıtlarının etkin olup olmadığı
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Konuşmanın oluşturulma tarihi ve saati
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      Konuşmanın son güncellenme tarihi ve saati
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="current_page" type="integer">
  Mevcut sayfa numarası
</ResponseField>

<ResponseField name="per_page" type="integer">
  Sayfa başına öğe sayısı
</ResponseField>

<ResponseField name="total" type="integer">
  Kriterlere uyan toplam konuşma sayısı
</ResponseField>

<ResponseField name="last_page" type="integer">
  Son sayfa numarası
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://app.whattalk.ai/api/user/conversations?type=widget&per_page=10" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://app.whattalk.ai/api/user/conversations?type=widget&per_page=10',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const data = await response.json();
console.log(data.data); // Array of conversations
```

```python Python
import requests

response = requests.get(
    'https://app.whattalk.ai/api/user/conversations',
    headers={'Authorization': 'Bearer YOUR_API_KEY'},
    params={
        'type': 'widget',
        'per_page': 10
    }
)

data = response.json()
print(data['data'])
```

## Konuşma Türleri

| Tür | Açıklama |
|-----|----------|
| `test` | Asistan test arayüzünden yapılan dahili test konuşmaları |
| `widget` | Web sohbet widget'ından gelen konuşmalar |
| `whatsapp` | WhatsApp Business konuşmaları |
| `api` | API aracılığıyla oluşturulan konuşmalar |

## Kullanım Alanları

- **Analitik Panosu**: Konuşma metriklerini ve trendleri görüntüleyin
- **CRM Entegrasyonu**: Konuşma verilerini müşteri veritabanınızla senkronize edin
- **Kalite İzleme**: Tür ve asistana göre konuşma hacimlerini inceleyin
- **Fatura İnceleme**: Kuruluşunuz genelinde konuşma maliyetlerini takip edin
