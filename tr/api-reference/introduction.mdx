WhatTalk API dokümantasyonuna hoş geldiniz. API'miz, WhatTalk platformu ile programatik olarak etkileşim kurmanızı sağlayarak potansiyel müşteriler oluşturmanıza ve yönetmenize, aramalar planlamanıza ve daha fazlasını yapmanıza olanak tanır.

## Temel URL

Tüm API istekleri aşağıdaki temel URL'ye yapılmalıdır:

```
https://app.whattalk.ai/api
```
